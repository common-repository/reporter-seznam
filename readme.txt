Tento plugin přidá do stránky meta tag pro ověření vlastníka webu při registraci stránky do Seznam Reporter (Webmaster tools)

== Instalace ==

Jak plugin nainstalovat

1. Nahrajte plugin do složky `/wp-content/plugins/29-reporter-seznam` nebo ho nainstalujte přímo přes Wordpress pluginy
2. Aktivujte plugin na obrazovce 'Pluginy'
3. Přejděte na obrazovku Pluginy->Seznam Reportér
4. Do políčka klíč zadejte hodnotu meta tagu vygenerovanou na stránkách [Seznam Webmaster tools](https://reporter.seznam.cz)
